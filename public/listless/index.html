Listless

Do more with less.

  1. What? Listless is a list app for iPhone, iPad and Mac that syncs using iCloud. One list, multiple devices.
  2. Why? Sometimes you just need a list. Add, reorder and complete items in a flash. Sync the list everywhere. Get on with your life.
  3. How (much)? Listless is free.

How much did you rely on coding agents…

Almost entirely. I created Listless as an experiment to see what I could accomplish using modern coding agents operating under human direction. I did my best to avoid producing A.I. slop. You can judge for yourself how well I did. Listless is open source software and the repository is available on GitHub.
